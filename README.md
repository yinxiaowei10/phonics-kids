# 自然拼读小课堂 Phonics Kids

一个面向儿童的英语自然拼读学习应用，帮助孩子像母语孩子一样掌握字母发音、拼读规则和基础词汇。

## 在线体验

直接在浏览器中打开 `index.html` 即可使用。

## 当前版本

**v0.1.0** — 独立项目搭建完成

## 功能

- **5 个学习级别**：The Alphabet、Short Vowels、Long Vowels、Consonant Blends、Letter Combinations
- **8 个学习单元 / Level**：循序渐进学习
- **4 种学习模式**：
  - 学习模式：学习字母发音、keyword 和例词
  - 听音辨音：听发音选择正确字母
  - 拼读练习：跟着拆分拼读单词
  - 听音写词：听发音写出单词
- **Web Speech API 朗读**：支持英文发音播放
- **响应式设计**：适配桌面和移动设备

## 数据来源

Level 1 数据来自 *Oxford Phonics World* 学生用书。Level 2-5 数据将逐步补充。
